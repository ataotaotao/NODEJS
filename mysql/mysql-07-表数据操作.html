表操作

insert

  • 插入完整的记录

    • insert into tb_name (column_list) valuoues(VALUE);
    • 咧名称可以省略，但是必须按照建表顺序逐一指定值。如果指定了值，那么表结构改变的话，上面那种无影响。

  • 插入记录的一部分 ， 指定列

    • 只向部分列中插入值，其他没有插入的列使用默认值。

  • 插入多条记录

    • insert into tbname(column_list) values (value_list1) , (value_list1) , (value_list1) ;
    • 插入多条记录可以使用这个，更快。

  • 插入另一个查询的结果

    • insert into xxxx select * from xxxx where (id > 5);

limit

限制查询记录的数量。

xxxxxxxxxx
limit [位置偏移量，] 行数
指定了位置偏移量就从指定的位置开始，不指定就从表中的第一行开始
行数返回记录的条数
select * from bookinfo limit 3;
从第一行开始，显示3条记录
select * from bookinfo limit 2,2;
从第三条记录开始后的两条图书信息。
mysql5.7可以使用limit 2 offset 2 实现同样的效果。

案例

xxxxxxxxxx
将图书信息按照库存进行分组，统计每组库存下的个数，然后按库存进行降序排序，并查看结果中的前四条记录。
​
select store,count(*) from bookinfo 
    group by store desc 
    limit 4;
